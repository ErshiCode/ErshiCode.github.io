##  [两数之和](https://leetcode.cn/problems/two-sum/description/?envType=study-plan-v2&envId=top-100-liked) <Badge text="简单" type="tip" vertical="middle" />
::: info 题目描述
给定一个整数数组 `nums` 和一个整数目标值 `target`，请你在该数组中找出 **和为目标值** `target` 的那 **两个** 整数，并返回它们的数组下标。

* 你可以假设每种输入 **只会对应一个答案**。
* 数组中同一个元素在答案里 **不能重复出现**。
* 你可以按任意顺序返回答案。
  :::

### 示例

```text
输入：nums = [2,7,11,15], target = 9
输出：[0,1]
解释：因为 nums[0] + nums[1] == 9 ，返回 [0, 1] 。

输入：nums = [3,2,4], target = 6
输出：[1,2]

输入：nums = [3,3], target = 6
输出：[0,1]
```

## 解析
枚举：
* 第一层 for 循环固定第一个数，第二层 for 循环依次枚举第二个数，暴力解法时间复杂度为 (O(N^2))

哈希表：
* 创建 HashMap 使用 key 存储数组元素，value 存储数组下标；
* 遍历数组，判断 map 中的 key 是否存在等于 目标值 - 当前元素 的值，如果存在直接返回 map 中 value 值和当前数组下标； 不存在则将数组元素和下标添加到 map 中；
* 遍历结束依旧没找到对应答案，返回数组空值；
> 哈希表相比于枚举只需实现一次遍历，将时间复杂度从 (O(N^2)) 降低到 O（N）

## 代码

::: tabs
@tab 哈希表 (O(N))
```java
 class Solution {
    public int[] twoSum(int[] nums, int target) {
        Map<Integer, Integer> map = new HashMap<>();
        for (int i = 0; i < nums.length; i++) {
            if (map.containsKey(target - nums[i])) {
                return new int[]{map.get(target - nums[i]), i};
            }
            map.put(nums[i],i);
        }
        return new int[]{};
    }
}
```
@tab 枚举 (O(N^2))
```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        int n = nums.length;
        for (int i = 0; i < n; i++) {
            for (int j = i + 1; j < n; j++) {
                if (nums[i] + nums[j] == target) {
                    return new int[]{i, j};
                }
            }
        }
        return new int[0];
    }
}
```

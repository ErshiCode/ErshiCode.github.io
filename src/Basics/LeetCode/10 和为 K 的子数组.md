## [和为 K 的子数组](https://leetcode.cn/problems/subarray-sum-equals-k/description/?envType=study-plan-v2&envId=top-100-liked)
::: info 题目描述
给你一个整数数组 `nums` 和一个整数 `k` ，请你统计并返回 *该数组中和为 `k` 的子数组的个数* 。
子数组是数组中元素的连续非空序列。
:::

## 示例

```text
输入：nums = [1,1,1], k = 2
输出：2
解释：[1,1] 与 [1,1] 为两种不同的子数组。

输入：nums = [1,2,3], k = 3
输出：2
解释：[1,2] 与 [3] 是和为 3 的子数组。
```

## 解析
**前缀和 + 哈希表：**
* 前缀和公式：num[i,x,x,,x,x,j]的和 pre[j] - pre[i - 1], 和为 k 的数组 pre[i - 1] = pre[j] - k;


## 代码
```java
class Solution {
    public int subarraySum(int[] nums, int k) {
        Map<Integer, Integer> map = new HashMap<>();
        map.put(0, 1);

        int sum = 0, res = 0;
        for (int num : nums) {
            sum += num;
            res += map.getOrDefault(sum - k, 0);
            map.put(sum, map.getOrDefault(sum, 0) + 1);
        }

        return res;
    }
}
```

## [最长连续序列](https://leetcode.cn/problems/longest-consecutive-sequence/description/?envType=study-plan-v2&envId=top-100-liked) 

::: info 题目描述
给定一个未排序的整数数组 `nums` ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。

* 请你设计并实现时间复杂度为 **$O(N)$** 的算法解决此问题。
  :::

### 示例

```text
输入：nums = [100, 4, 200, 1, 3, 2]
输出：4
解释：最长数字连续序列是 [1, 2, 3, 4]。它的长度为 4。

输入：nums = [0,3,7,2,5,8,4,6,0,1]
输出：9

输入：nums = [1,0,1,2]
输出：3
```

## 解析
**哈希表法：**
* 遍历数组元素添加到 set 当中进行去重，定义最长连续序列长度值
* 若找到连续序列需先找到开头元素，遍历集合元素找到不包含当前元素 - 1 的元素即为连续序列开头元素
* 记录当前元素值、元素长度在 set 中查找连续序列，最后与最长连续序列值进行比较

## 代码

:::tabs
@tab
```java
class Solution {
    public int longestConsecutive(int[] nums) {
        Set<Integer> set = new HashSet<>();
        for (int num : nums) {
            set.add(num);
        }

        int res = 0;
        for (int num : set) {
            if (set.contains(num - 1)) {
                continue;
            }

            int curNum = num;
            int curLen = 1;
            while (set.contains(curNum + 1)) {
                curNum++;
                curLen++;
            }

            res = Math.max(res, curLen);
        }

        return res;
    }
}
```

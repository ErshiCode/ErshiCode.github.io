## [三数之和](https://leetcode.cn/problems/3sum/?envType=study-plan-v2&envId=top-100-liked) <Badge text="中等" type="warning" vertical="middle" />
::: info 题目描述
给你一个整数数组 `nums`，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j`、`i != k` 且 `j != k`，同时还满足 `nums[i] + nums[j] + nums[k] == 0`。
请你返回所有和为 0 且不重复的三元组。
* 答案中不可以包含重复的三元组。
:::

### 示例
```text
输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]
解释：
nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0
nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0
不同的三元组是 [-1,0,1] 和 [-1,-1,2] 。

输入：nums = [0,1,1]
输出：[]
解释：唯一可能的三元组和不为 0 。

输入：nums = [0,0,0]
输出：[[0,0,0]]
解释：唯一可能的三元组和为 0 。
```

### 解析
排序 + 双指针：
* 利用 sort函数 对数组进行排序，通过 for 循环遍历数组固定好第一个元素（nums[i]），判断数组元素是否大于0、存在相同元素；
* 取第一个元素右边值为左指针、最后一个元素为右指针，left < right 时比较num[i]、nums[left]、nums[right] 三数之和与 0 的关系；
* 当 == 0 时将三数添加到 res 中并且因题目不要求重复元素，所以移动左右指针时需要进行去重；
* 当 > 0 时因为已经通过 sort 函数排序，所以需要将右指针左移；
* 当 < 0 时因为已经通过 sort 函数排序，所以需要将左指针右移；

```java
class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> res = new ArrayList<>();
        Arrays.sort(nums);
        for (int i = 0; i < nums.length - 2; i++) {
            if (nums[i] > 0) break;
            if (i > 0 && nums[i] == nums[i - 1]) continue;
            int left = i + 1, right = nums.length - 1;
            while (left < right) {
                int sum = nums[left] + nums[right] + nums[i];
                if ( sum == 0) {
                    res.add(Arrays.asList(nums[i]), nums[left], nums[right]);
                    while (left < right && nums[left] == nums[left + 1]) left++;
                    while (left < right && nums[right] == nums[right - 1]) right--;
                    left++; right--;
                } else if (sum > 0) {
                    right--;
                } else {
                    left++;
                }
            }
        }
        return res;
    }
}
```

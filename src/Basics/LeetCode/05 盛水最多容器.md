## [盛最多水的容器](https://leetcode.cn/problems/container-with-most-water/description/?envType=study-plan-v2&envId=top-100-liked)
::: info 题目描述
给定一个长度为 `n` 的整数数组 `height`。有 `n` 条垂线，第 `i` 条线的两个端点是 `(i, 0)` 和 `(i, height[i])`。

找出其中的两条线，使得它们与 `x` 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

* **说明：** 你不能倾斜容器。
  :::

## 示例
```text
输入：[1,8,6,2,5,4,8,3,7]
输出：49 
解释：图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49。
(即：高度为 min(8, 7) = 7，宽度为 8 - 1 = 7，面积 = 7 * 7 = 49)

输入：height = [1,1]
输出：1
```

## 解析
**双指针 + 贪心：**
* 初始化 left 指向数组首位 right 指向数组末尾
* 遍历数组，right - left 为长度 数组中 left right 较小的一方为宽度，相乘计算面积
* 根据贪心算法，计算完当前面积之后移动较小的一侧继续计算面积，以求最大值


## 代码

```java
class Solution {
    public int maxArea(int[] height) {
        int left = 0, right = height.length - 1, res = 0;
        for (int i = 0; i < height.length; i++) {
            int area = (right - left) * Math.min(height[left], height[right]);
            res = Math.max(res, area);
            if (height[left] < height[right]) {
                left++;
            } else {
                right--;
            }
        }
        return res;
    }
}
```
